Estrutura criada com .NET 9 (net9.0).

Como rodar:
1. Garanta que o .NET 9 SDK esteja instalado.
2. Na raiz do diretório, rode:
   dotnet build
3. Para executar o runner (console que demonstra todos os módulos):
   dotnet run --project tests/Runner

Observação: cada módulo (Importacao, Pedidos, Pagamento, Sync) está em seu próprio projeto dentro de `src/`.
